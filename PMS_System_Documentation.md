# RMS Resident Management System Documentation

## 1. Technology Stack

-   MongoDB
-   Express.js
-   React.js
-   Node.js (MERN Stack)

## 2. System Roles

-   Admin (Full Control)
-   Special Employee (Admin Assistant)
-   Employee
-   Resident

## 3. Admin Features

### Resident Management

-   Add/Register Residents
-   Update Resident Information
-   Deactivate Residents
-   View Resident Profiles
-   Manage Dependents
-   Approve Certificates

### Employee Management

-   Add/Register Employees
-   Update Employee Info
-   Classify Employees by Job Category
-   Assign/Reassign Jobs
-   Monitor Job Progress

### Special Employee Management

-   Add/Remove Special Employees
-   Set Permissions

### Job & Task Management

-   Create Jobs
-   Set Category & Priority
-   Assign Jobs
-   Track Progress
-   Mark Completed

### Request & Complaints

-   View Maintenance Requests
-   View Complaints
-   Convert Requests to Jobs

### Communication

-   Send Notifications
-   Announcements

### Digital ID System

-   Generate Digital ID Card
-   QR Verification
-   Approve ID Renewals
-   Certificate Verification
-   Generate PDF ID Information

### System Controls

-   Global Search
-   Reports
-   Settings
-   Full Dashboard
